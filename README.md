# vue-chaos
A simple yet chaotic component to introduce chaos in your Vue app

#### Usage
``
## Project setup
Install
```
npm i vue-chaos
```
Import & use
```vue
<template>
       <VueChaos>
          <MyOtherComponent />
        </VueChaos>
</template>
<script>
    import VueChaos from 'vue-chaos';
    ...
    export default {
      ...
      components:{
        VueChaos
      }
    }
</script>
```
